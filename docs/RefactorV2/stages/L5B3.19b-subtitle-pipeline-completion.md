## 3. 风险监测

### 3.1 attrs 体积膨胀
translationTexts 是 JSON.stringify(Record<lang, fullText>)。一个 60 分钟视频 transcript ~2-3KB,翻译成 5 种语言 ~12-15KB attrs 大小。**仍在合理范围**(对比 image base64 几 MB)。后续如出问题再考虑 gzip。

### 3.2 youtube-transcript 失败模式
默认捕获,降级到 `error: 详情`。视频禁字幕 / API 限流 / 网络错误都会进 catch。**fetchYouTubeTranscript 实现已覆盖**。

### 3.3 翻译 Tab 数量无上限
对齐 V1。如反复切语言,Tab 越多 attrs 越大。**接受现状**,后续如真出问题加最大数量限制。
