## 1. 实际改动 vs 设计 — 偏差点

无显著偏差,完全照 v0.2 设计落:

- W5-A View 边界全程合规:DictionaryPanel + learning-integration 都通过
  `requireCapabilityApi<LearningApi>('learning')` / `<TextEditingApi>('text-editing')`
  间接路由,view 不直 `import @drivers/*` / `import @capabilities/learning`(运行时函数);
  types 是类型 only 纯擦除,允许直 import。
- vocab-highlight plugin 在 driver 层(W5-A 允许 driver 直 import capability),
  module-level `vocabDefs` Map(tooltip 释义)+ `updateVocabDefs` export,
  通过 `setVocabWords(entries)` 一次性更新所有 PM instance(`instanceRegistry.getAll()`)。
- tooltip 用 plugin 自创建 DOM 挂 `document.body`(不走 popup-registry,
  避免 hover singleton 冲突 — 决策 Q5=自管 DOM)。
- DictionaryPanel 用模块级 `pendingMode/pendingText/pendingContext` + `setPanelInitial`
  传递 popup 初始参数(对齐 V2 已有 popup 模式 — 决策 Q3=模块级状态)。

**P1 / P2 审计修正全部落地**:
- P1:DictionaryPanel.tsx + learning-integration.ts 全用 `requireCapabilityApi`
  代替 `import { ... } from '@capabilities/learning'`(运行时函数)。
- P2:类型名称统一 `DictionaryLookupResult`(对齐 20a 后端定义)。

## 3. 用户验收清单(请按顺序点检)

### 3.1 启动 + 基础(冒烟)

1. `npm start`启动应用 → 控制台无 capability 注册 / install-coverage 报错
2. 打开任一 NoteView(笔记页面)→ 编辑器正常出现,光标可输入
3. F12 console:无未捕获异常

### 3.2 右键查词 / 翻译

4. 在笔记里输入英文段落,例如 `The dictionary panel should appear when clicked.`
5. 双击选中 `dictionary` → 右键 → 应看到菜单底部含两项:
   - 📖 查词
   - 🌐 翻译
6. 点 **📖 查词** → dictionary popup 弹在选区下方,显:
   - 词头 `dictionary` + 音标(若有)+ TTS 喇叭 🔊
   - 英文释义(macOS Dictionary 或 Wiktionary 兜底)
   - 中文翻译(Google translate)
   - 蓝色按钮 "添加到生词本"
7. 点喇叭 🔊 → TTS 播放(B3.20a CSP 已加 blob:)
8. 点 "添加到生词本" → 按钮文案 / 状态变为已添加(disabled / 灰)
9. Esc → popup 关闭
10. 选中整句 → 右键 **🌐 翻译** → 显原文 + 翻译,无释义 / 无 "添加到生词本"
   (translate 模式简化 UI,加生词本对句子无意义)

### 3.3 vocab-highlight 高亮

11. 步骤 8 加完 `dictionary` 后 → 该笔记里所有 `dictionary` 应黄色底虚下划线
12. 鼠标 hover 高亮词 → 出 `.vocab-tooltip` 浮层,显该词 + 简释义 + TTS
13. tooltip 上点 🔊 → TTS 播放该词
14. 鼠标移开高亮 → tooltip 消失
15. 同一 word 在第二个 NoteView(打开另一笔记)也应该高亮 — `setVocabWords`
    遍历所有 `instanceRegistry.getAll()` 推 decoration

### 3.4 vocab Tab(生词本管理)

16. 再打开 dictionary popup(任意查词)→ 切顶 Tab "生词本" →
    列表显已加的 entry(单词 + 简释义)
17. 搜索框输 `dic` → 实时过滤
18. 点 entry → 切回 "查词" Tab,以该词重查
19. 点 entry 右侧 ✕ → entry 移除,高亮即时消失(onVocabChanged 触发推流)
20. 列表为空时显灰色提示 "暂无生词"

### 3.5 W5-A 边界自检

21. eslint --max-warnings 0 全过(已确认)
22. install-coverage 自检通过 — 启动时控制台无 missing 警告
