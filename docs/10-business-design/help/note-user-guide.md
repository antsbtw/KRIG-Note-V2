# KRIG Note 使用手册

> 这篇手册本身就是一篇 Note。你可以直接在手册中操作、编辑、尝试——边看边练，学以致用。

---

## 第一章：快速入门

### 1.1 创建你的第一篇 Note

在左侧导航栏（NavSide）中，你可以通过以下方式新建一篇 Note：

- 点击导航栏顶部的 **＋** 按钮
- 右键点击空白区域，选择「新建笔记」

新建的 Note 会自动出现在导航栏中，光标直接定位到标题处，你可以立即输入标题。

### 1.2 基本输入与编辑

Note 的编辑区域就是你正在看的这个区域。每一段文字、每个标题、每个代码块……都是一个独立的「块」（Block）。

**基本操作：**

| 操作 | 快捷键 |
|------|--------|
| 撤销 | `Cmd + Z` |
| 重做 | `Cmd + Shift + Z` 或 `Cmd + Y` |
| 剪切 | `Cmd + X` |
| 复制 | `Cmd + C` |
| 粘贴 | `Cmd + V` |
| 纯文本粘贴 | `Cmd + Shift + V` |
| 全选 | `Cmd + A` |

**试一试：** 在下方空行中输入一些文字，然后用 `Cmd + Z` 撤销你的输入。

### 1.3 斜杠菜单 `/`

斜杠菜单是你最常用的工具。在空行的开头输入 `/`，会弹出一个菜单，列出所有可插入的内容块类型。

**使用方法：**

1. 在空行开头输入 `/`
2. 继续输入关键词来过滤（如 `/code`、`/h1`、`/table`）
3. 用 `↑` `↓` 方向键选择
4. 按 `Enter` 确认插入
5. 按 `Esc` 关闭菜单

**全部斜杠命令一览：**

**基础块（Basic）：**

| 命令 | 图标 | 作用 |
|------|------|------|
| `/paragraph` | ¶ | 普通段落 |
| `/h1` | H1 | 一级标题 |
| `/h2` | H2 | 二级标题 |
| `/h3` | H3 | 三级标题 |
| `/code` | </> | 代码块（支持 `/code javascript` 直接指定语言） |
| `/bullet` | • | 无序列表 |
| `/number` | 1. | 有序列表 |
| `/task` | ☐ | 任务清单 |
| `/quote` | ❝ | 引用块 |
| `/toggle` | ▶ | 折叠块 |
| `/math` `/latex` | ∑ | 数学公式块 |
| `/callout` | 💡 | 提示框 |
| `/table` | ▦ | 表格 |
| `/hr` `/divider` | — | 水平分割线 |
| `/inline math` | ∑ | 行内数学公式 |

**布局（Layout）：**

| 命令 | 图标 | 作用 |
|------|------|------|
| `/frame` | ▢ | 彩色边框块 |
| `/column` (two) | ▥ | 两栏布局 |
| `/column` (three) | ▥ | 三栏布局 |

**代码图表（Code）：**

| 命令 | 图标 | 作用 |
|------|------|------|
| `/mermaid` | ◇ | Mermaid 流程图/图表 |
| `/canvas` | 📄 | JavaScript 交互画布 |

**多媒体（Media）：**

| 命令 | 图标 | 作用 |
|------|------|------|
| `/image` | 🖼 | 图片 |
| `/video` | 🎬 | 视频（YouTube/Vimeo/本地） |
| `/audio` | 🎵 | 音频 |
| `/tweet` `/x` | 𝕏 | X (Twitter) 帖子嵌入 |
| `/file` `/attach` | 📎 | 文件附件 |
| `/ref` `/external` | 🔗 | 外部引用卡片（文件/URL） |
| `/html` | </> | HTML 实时预览块 |

**试一试：** 在下方空行输入 `/` 并浏览一下菜单中的所有选项。

## 第三章：标题与文档结构

### 3.1 标题层级

Note 支持三级标题，用于组织文档结构：

| 标题 | 快捷键 | 斜杠命令 |
|------|--------|----------|
| 一级标题 H1 | `Cmd + Alt + 1` | `/h1` |
| 二级标题 H2 | `Cmd + Alt + 2` | `/h2` |
| 三级标题 H3 | `Cmd + Alt + 3` | `/h3` |
| 转回正文 | `Cmd + Alt + 0` | — |

**切换行为：** 如果当前块已经是 H1，再按 `Cmd + Alt + 1` 会把它转回普通段落（toggle）。

**试一试：** 把光标放在下面这行，按 `Cmd + Alt + 2` 把它变成二级标题：

这行可以变成标题

### 3.2 标题折叠

每个标题右侧有一个折叠按钮（⌃/⌄）。点击后，该标题下属的所有内容会被折叠隐藏，只显示标题本身。

- 折叠 H1 会隐藏该 H1 下的所有内容（直到下一个 H1）
- 折叠 H2 会隐藏该 H2 下的内容（直到下一个 H1 或 H2）
- H3 同理

折叠状态会被保存，下次打开 Note 时仍然保持。

你也可以通过块手柄菜单（⠿）→「折叠/展开」来操作。

### 3.3 目录导航

标题会自动生成目录（TOC），显示在 Note 的侧边。你可以点击 TOC 中的标题快速跳转到对应位置。

### 3.4 水平分割线

输入 `---`（三个连字符）后按空格或回车，会自动转换为水平分割线。也可以用 `/hr` 插入。

## 第四章：列表与任务

### 4.1 无序列表

**创建方式：**
- 输入 `- ` 或 `* ` 或 `+ `（注意后面有空格），自动转为无序列表
- 或使用 `/bullet-list`

**操作：**
- `Enter`：在当前项后新建一项
- `Enter`（空项上）：退出列表，转为普通段落
- `Tab`：增加缩进，变为子项
- `Shift + Tab`：减少缩进

**试一试：** 在下方输入 `- ` 开始一个列表，然后用 Tab 创建嵌套结构：

### 4.2 有序列表

**创建方式：**
- 输入 `1. `（数字加点加空格），自动转为有序列表
- 或使用 `/ordered-list`

编号会自动递增，调整顺序时会自动重新编号。

### 4.3 任务清单

**创建方式：**
- 输入 `[ ] `（方括号加空格），自动转为任务项
- 或使用 `/task-list`

**任务清单的特殊功能：**
- 点击复选框 ☐/☑ 切换完成状态
- 完成后文字会自动添加删除线，并记录完成时间戳
- 点击时间标签可以设置截止日期（deadline）
- 超过截止日期的任务会显示「逾期」标记

**试一试：** 在下方创建几个任务，勾选一些，并给其中一个设置截止日期：

### 4.4 缩进与嵌套

所有块类型都支持缩进（最多 10 级）：

| 操作 | 快捷键 |
|------|--------|
| 增加缩进 | `Tab` |
| 减少缩进 | `Shift + Tab` |
| 首行文字缩进 | `Shift + Cmd + I` |

缩进特别适合在列表中创建层级结构。

## 第五章：引用、Callout 与容器块

### 5.1 引用块

**创建方式：**
- 输入 `> `（大于号加空格），自动转为引用块
- 或使用 `/blockquote`

引用块显示为左侧带竖线的缩进段落，适合引用他人的话或强调内容。

> 这是一个引用块的示例。你可以在引用块内继续输入多行内容。

### 5.2 Callout 提示框

使用 `/callout` 插入一个 Callout 提示框。它是带有 emoji 图标的醒目提示区域。

**emoji 选择：** 点击 Callout 左侧的 emoji 图标，可以从 10 个预设中选择：

💡 提示 · ⚠️ 警告 · ❌ 错误 · ✅ 成功 · ℹ️ 信息 · 🔥 重要 · 📌 固定 · 💬 讨论 · 🎯 目标 · ⭐ 收藏

Callout 内部可以输入任意内容，适合用来标注重点信息。

**试一试：** 在下方用 `/callout` 插入一个提示框，然后切换它的 emoji 图标。

### 5.3 折叠块 Toggle

使用 `/toggle-list` 插入一个可展开/收起的内容区域。

- 点击 ▶ 展开内容
- 点击 ▼ 收起内容
- Toggle 的标题始终可见，内部内容可以是任意类型的块

Toggle 适合用来组织「常见问题」、「详细解释」等按需展开的内容。

**试一试：** 用 `/toggle-list` 创建一个 Toggle，在里面写一些内容，然后点击箭头收起它。

### 5.4 Frame 彩色边框块

使用 `/frame-block` 或通过块手柄菜单（⠿）→「Frame」来给块添加彩色边框。

**可选颜色（6 种）：** 蓝、红、绿、黄、紫、青

**边框样式：** 实线（solid）或双线（double）

**高级用法：Frame 分组**
你可以将多个相邻的块链接到同一个 Frame 分组（groupId），它们会共享同一个连续的边框，形成视觉上的整体区域。

## 第六章：布局

### 6.1 分栏布局

使用 `/column2` 或 `/column3` 插入两栏或三栏布局。

**操作：**
- 每一栏内部可以放入任意类型的块
- 拖拽栏间的分隔线可以调整列宽（最小 20%）
- 工具栏提供「添加列」和「删除列」按钮
- 每列可以设置垂直对齐方式：顶部 / 居中 / 底部

**试一试：** 用 `/column2` 插入一个两栏布局，在左栏写一段文字，右栏插入一个列表。

### 6.2 表格

使用 `/table` 插入一个 3×3 的初始表格。

**表格操作：**
- 在单元格中直接输入内容
- `Tab`：跳到下一个单元格
- 拖拽列边框可以调整列宽
- 单元格内可以包含富文本（加粗、链接、代码等）
- 支持合并单元格（colspan/rowspan）

**试一试：** 用 `/table` 插入一个表格，输入一些数据，然后试试用 Tab 键在单元格间跳转。

### 6.3 对齐与缩进

通过块手柄菜单（⠿）→「格式」可以设置文本块的对齐方式：

| 对齐 | 效果 |
|------|------|
| 左对齐 | 文字靠左（默认） |
| 居中 | 文字居中 |
| 右对齐 | 文字靠右 |
| 两端对齐 | 文字两端对齐 |

首行缩进可以通过 `Shift + Cmd + I` 切换开关。

## 第七章：代码与图表

### 7.1 代码块

使用 `/code` 插入代码块。

**功能：**
- **语言选择**：点击左上角的语言标签，搜索并切换语言（支持 20+ 种语言：JavaScript、TypeScript、Python、Rust、Go、Java、C/C++、HTML、CSS、JSON、YAML、SQL、Bash、LaTeX、Mermaid 等）
- **语法高亮**：基于 CodeMirror，深色主题
- **行号显示**：自动编号
- **复制按钮**：一键复制代码，成功后显示绿色 ✓
- **导出 PNG**：将代码块截图为 2x 高清 PNG 图片
- **全屏查看**：展开为全屏视图，支持平移和缩放

**试一试：** 用 `/code` 插入一个代码块，选择 JavaScript 语言，输入一段代码，然后试试复制和导出 PNG 功能。

### 7.2 Mermaid 图表

使用 `/mermaid` 插入 Mermaid 图表块。

**三种查看模式：**
- **代码模式**：只显示 Mermaid 源码编辑器
- **分屏模式**：左边代码，右边实时预览
- **预览模式**：只显示渲染后的图表

**导出：** 支持导出为 SVG 或 PNG 格式。

**全屏：** 支持全屏查看，可拖拽平移和滚轮缩放。

**试一试：** 用 `/mermaid` 插入一个图表块，输入以下 Mermaid 代码看看效果：

```
graph TD
    A[开始] --> B{条件判断}
    B -->|是| C[执行操作]
    B -->|否| D[跳过]
    C --> E[结束]
    D --> E
```

### 7.3 Canvas 画布

使用 `/canvas` 插入一个 JavaScript 画布块。你可以在 CodeMirror 编辑器中编写 JavaScript 代码来绑定交互式的 Canvas 绘图。

### 7.4 数学公式块

使用 `/latex` 或在空行输入 `$$` 来插入一个块级数学公式。

公式使用 KaTeX 渲染，支持完整的 LaTeX 数学语法。

**示例：**
```
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
```

**试一试：** 用 `/latex` 插入一个公式块，输入上面的高斯积分公式。

### 7.5 数学可视化

使用 `/math-visual` 插入一个交互式数学函数可视化块。

**功能：**
- 输入 LaTeX 格式的函数表达式，自动绘制函数图像
- **参数滑块**：为函数中的参数添加可拖动滑块，实时观察图像变化
- **黎曼和工具**：可视化积分的矩形逼近
- **积分/切线/法线**：交互式数学分析工具
- **全屏模式**：展开为全屏交互视图

## 第八章：多媒体

### 8.1 图片

使用 `/image` 插入图片块。

**插入方式：**
- 从文件系统上传
- 粘贴 URL 嵌入网络图片
- 直接从剪贴板粘贴图片（`Cmd + V`）

**操作：**
- **对齐**：左对齐 / 居中 / 右对齐
- **缩放**：拖拽左右两侧的手柄调整图片大小
- SVG 图片会以内联方式渲染，支持 CSS 变量

### 8.2 视频

使用 `/video` 插入视频块。

**支持的来源：**
- YouTube 链接
- Vimeo 链接
- 本地视频文件

**高级功能：**
- **字幕系统**：支持带时间戳的字幕，可解析 YouTube 字幕
- **YouTube 字幕导入**：自动获取 YouTube 视频的字幕文本
- **记忆回放模式（Memory Playback）**：基于间隔重复原理，按设定的时间间隔自动回放标记的片段
- **词汇时间轴**：在时间轴上标记学习过的词汇位置
- **翻译支持**：字幕翻译功能
- **全屏查看**

### 8.3 音频

使用 `/audio` 插入音频播放器。

**功能：**
- HTML5 音频播放控件
- 支持下载（网络音频）
- 字幕/标注支持
- 可编辑标题

### 8.4 Tweet 嵌入

使用 `/tweet` 插入一个 Tweet 展示块。粘贴 Tweet 数据后，会渲染为卡片形式，点击可打开原始 Tweet。

### 8.5 HTML 块

使用 `/html` 插入一个 HTML 渲染块。

**功能：**
- 左侧代码编辑器（带语法高亮）
- 右侧实时预览
- 支持导出为 PNG 或 SVG

适合嵌入自定义的可视化内容、小组件等。

## 第九章：文件与引用

### 9.1 文件块

使用 `/file` 插入一个文件块。

**操作：**
- 点击上传或选择本地文件
- 文件会嵌入到 Note 中，显示为预览卡片
- 提供「打开」和「下载」按钮
- 自动识别文件 MIME 类型

### 9.2 行内文件链接

在文本中可以插入行内文件链接，显示为 📎 文件名 的芯片形式。

**操作：**
- 点击：用默认应用打开文件
- 右键菜单：
  - 打开文件
  - 在 Finder 中显示
  - 复制文件路径

### 9.3 外部引用卡片

使用 `/external-ref` 插入一个外部引用卡片。

**支持两种引用类型：**
- 📁 **文件引用**：指向本地文件，显示文件路径，提供「打开」和「在 Finder 中显示」按钮
- 🌐 **网页引用**：指向网页 URL，显示域名，提供「打开」按钮

### 9.4 笔记间链接

你可以在文本中插入指向其他 Note 的链接。

**创建方式：** 按 `Cmd + K`，在链接面板的「笔记」标签页中搜索目标 Note，还可以选择具体的标题锚点。

链接格式为 `krig://block/{noteId}/{anchor}`，点击后会跳转到目标 Note 的对应位置。

你也可以通过块手柄菜单（⠿）→「复制链接」来获取当前块的链接地址，分享给其他人或在其他 Note 中引用。

### 9.5 页面锚点

使用 `/page-anchor` 插入一个页面锚点。锚点是不可见的标记，用于为块级位置提供精确的跳转目标。

## 第十章：块操作

### 10.1 拖拽排序

每个块左侧有一个手柄图标（⠿），将鼠标悬停在块上时显示。

**拖拽移动：**
1. 按住 ⠿ 手柄
2. 拖动到目标位置
3. 松开鼠标放下

拖拽时会出现一条蓝色指示线，标识放置位置。拖拽可以在容器之间移动（如从一栏拖到另一栏，从列表外拖入列表内）。

### 10.2 块选择与多选

- **单选：** 点击块左侧的 ⠿ 手柄，选中整个块（高亮显示）
- **多选：** 按住 `Cmd` 再点击其他块的手柄，可以选中多个块
- **操作选中的块：**
  - `Delete` / `Backspace`：删除所有选中的块
  - `Cmd + X`：剪切所有选中的块
  - `Cmd + C`：复制所有选中的块

### 10.3 块手柄菜单 ⠿

点击 ⠿ 手柄（不拖拽），会打开块操作菜单：

| 菜单项 | 说明 |
|--------|------|

## 第十一章：智能粘贴

### 11.1 Cmd + V 智能粘贴

KRIG Note 的粘贴功能不是简单的文本粘贴——它会智能识别剪贴板内容的来源，并进行最优的格式转换。

**自动识别的来源：**

| 来源 | 处理方式 |
|------|----------|
| Microsoft Word | 解析 Word HTML，保留标题、列表、表格等结构 |
| Excel | 解析表格数据，转为 Note 表格 |
| Notion | 识别 Notion 格式，保留块结构 |
| VS Code | 识别代码格式，自动创建代码块并设置语言 |
| AI 对话（ChatGPT 等） | 解析 Markdown 格式，保留代码、列表等 |
| 网页 | 解析 HTML，转换为 Markdown 再导入 |
| 纯文本 | 直接插入为文本段落 |

### 11.2 Cmd + Shift + V 纯文本粘贴

如果你不想要格式转换，按 `Cmd + Shift + V` 会跳过所有 HTML 处理，只粘贴纯文本。

适用于：你只想要文字内容，不需要保留原始格式的场景。

### 11.3 图片/媒体粘贴

如果剪贴板中包含图片，`Cmd + V` 会自动检测并在光标位置创建图片块。

这在截图后直接粘贴到 Note 中非常方便——截图 → 切换到 Note → `Cmd + V`，图片就插入了。

## 第十二章：AI 与学习工具

### 12.1 Ask AI

选中内容后，通过右键菜单或块手柄菜单中的「🤖 Ask AI」可以调用 AI。

**使用流程：**
1. 选中想要处理的内容（或通过块手柄选中整个块）
2. 点击「Ask AI」
3. 在弹出的面板中输入指令（如「翻译成英文」「总结要点」「改写为正式语气」）
4. 选择 AI 服务
5. 按 `Enter` 发送
6. AI 的结果会插入到文档中

### 12.2 Thought 批注

Thought（思考批注）是 KRIG Note 的特色功能，让你可以在阅读和学习时记录即时想法。

**创建方式：**
- 快捷键 `Cmd + Shift + M`
- 右键菜单 →「Thought」
- 块手柄菜单 →「Thought」

**批注类型：**
- **行内批注**：附着在选中的文字上，类似高亮批注
- **块级批注**：附着在整个块上

Thought 会与外部的 Thought 系统关联，你可以在 Thought 视图中集中查看和管理所有批注。

### 12.3 翻译与词典

**翻译：** 选中文字后，右键菜单 →「翻译」，会将选中内容发送到翻译服务。

**词典查词：** 选中一个单词后，右键菜单 →「词典」，会显示该单词的释义。

这两个功能在阅读外文资料时非常实用。

### 12.4 词汇高亮与视频学习

在视频块（`/video`）中，KRIG Note 提供了专门的语言学习工具：

- **词汇时间轴标记**：在视频时间轴上标记你学习过的词汇出现的位置
- **记忆回放模式**：基于间隔重复（Spaced Repetition）原理，自动回放标记的片段以加强记忆
