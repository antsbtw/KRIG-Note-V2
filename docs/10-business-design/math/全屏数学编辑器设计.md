# 全屏数学编辑器设计

## 一、设计动机

### 当前 mathBlock 的局限

| 场景 | inline 编辑 | 全屏编辑 |
|------|:-----------:|:--------:|
| 单行公式 `E=mc^2` | 够用 | 不需要 |
| 多行对齐推导 `align` | 编辑框太窄 | 左右分屏舒适 |
| 大型矩阵 | 放不下 | 完整显示 |
| 推导链（多步骤） | 上下文碎片化 | 连续数学环境 |
| 公式+注释混排 | 混在笔记中 | 专注数学表达 |

**核心价值**：知识表达的质量取决于编辑环境的质量。复杂数学推导需要专注的、宽敞的编辑空间。

## 二、架构复用

全屏数学编辑器与 Mermaid 全屏编辑器共享框架：

```
┌─────────────────────────────────────────────┐
│ Toolbar                                     │
│ [符号面板 ∨] [环境 ∨] [−] 100% [+] [⛶] [×] │
├──────────────────────┬──┬───────────────────┤
│                      │  │                   │
│   LaTeX 编辑区       │分│   KaTeX 实时渲染   │
│   (CodeMirror)       │隔│                   │
│                      │线│                   │
│                      │  │                   │
├──────────────────────┴──┴───────────────────┤
│ 状态栏：✓ 渲染成功 / ✗ 错误信息              │
└─────────────────────────────────────────────┘
```

### 可复用的模块

| 模块 | Mermaid 已有 | Math 直接复用 |
|------|:-----------:|:------------:|
| 全屏 overlay 框架 | ✅ | ✅ |
| CodeMirror 编辑器 | ✅ | ✅ (换语法高亮) |
| 分屏拖拽 + 比例记忆 | ✅ | ✅ |
| 缩放控制条 | ✅ | ✅ |
| 状态栏错误提示 | ✅ | ✅ |
| 关闭同步回 ProseMirror | ✅ | ✅ |
| 导出 PNG/SVG | ✅ | ✅ (KaTeX → SVG) |

### Math 专有模块

| 模块 | 说明 |
|------|------|
| LaTeX 语法高亮 | CodeMirror 的 @codemirror/lang-latex 或自定义 StreamLanguage |
| KaTeX 渲染器 | 替换 mermaid.render → katex.renderToString |
| 符号面板 | LaTeX 常用符号快速插入 |
| 环境模板 | align, cases, matrix, pmatrix 等骨架插入 |

## 三、Toolbar 设计

### 3.1 符号面板（下拉面板）

点击 `符号` 按钮展开分类符号面板：

```
┌─────────────────────────────────────┐
│ 希腊字母    运算符    关系符    箭头  │  ← Tab 切换
├─────────────────────────────────────┤
│ α β γ δ ε ζ η θ ι κ λ μ            │
│ ν ξ π ρ σ τ υ φ χ ψ ω              │
│ Γ Δ Θ Λ Ξ Π Σ Φ Ψ Ω               │
└─────────────────────────────────────┘
```

**分类**：
- **希腊字母**：`\alpha` `\beta` `\gamma` ... `\Gamma` `\Delta` ...
- **运算符**：`\sum` `\prod` `\int` `\oint` `\partial` `\nabla` `\infty` ...
- **关系符**：`\leq` `\geq` `\neq` `\approx` `\equiv` `\sim` `\in` `\subset` ...
- **箭头**：`\to` `\gets` `\leftrightarrow` `\Rightarrow` `\Leftrightarrow` ...
- **装饰**：`\hat{}` `\bar{}` `\vec{}` `\dot{}` `\tilde{}` `\overline{}` ...
- **括号**：`\left(` `\right)` `\left[` `\right]` `\left\{` `\right\}` ...

**交互**：点击符号 → 插入对应 LaTeX 命令到光标位置

### 3.2 环境模板（下拉选择）

| 模板 | 插入内容 |
|------|---------|
| Align | `\begin{aligned}\n  a &= b \\\\\n  c &= d\n\end{aligned}` |
| Cases | `\begin{cases}\n  x & \text{if } x > 0 \\\\\n  -x & \text{otherwise}\n\end{cases}` |
| Matrix | `\begin{pmatrix}\n  a & b \\\\\n  c & d\n\end{pmatrix}` |
| Fraction | `\frac{numerator}{denominator}` |
| Sum | `\sum_{i=1}^{n} a_i` |
| Integral | `\int_{a}^{b} f(x) \, dx` |
| Limit | `\lim_{x \to \infty} f(x)` |

### 3.3 完整 Toolbar 布局

```
[符号 ∨] [环境 ∨] |  spacer  | [↓ PNG] [📋 PNG] | [−] 100% [+] [⛶] | [×]
```

## 四、预览区设计

### 4.1 渲染方式

```typescript
katex.render(latex, container, {
  displayMode: true,
  throwOnError: false,
  output: 'htmlAndMathml',  // 无障碍支持
  trust: true,              // 允许 \url 等
  macros: {},               // 用户自定义宏（未来）
});
```

### 4.2 多行推导渲染

LaTeX 源码：
```latex
\begin{aligned}
  \nabla \cdot \mathbf{E} &= \frac{\rho}{\epsilon_0} \\
  \nabla \cdot \mathbf{B} &= 0 \\
  \nabla \times \mathbf{E} &= -\frac{\partial \mathbf{B}}{\partial t} \\
  \nabla \times \mathbf{B} &= \mu_0 \mathbf{J} + \mu_0 \epsilon_0 \frac{\partial \mathbf{E}}{\partial t}
\end{aligned}
```

预览区居中渲染，字号 1.4em，滚动浏览。

### 4.3 错误处理

KaTeX 返回的错误比 Mermaid 更精确：
- `KaTeX parse error: Unknown symbol: \foo at position 15`
- 状态栏显示错误信息 + 位置

## 五、LaTeX 语法高亮

### 方案选择

| 方案 | 优劣 |
|------|------|
| `@codemirror/legacy-modes` 的 stex | 现成，但可能不完整 |
| 自定义 StreamLanguage | 完全控制，类似 mermaid-lang.ts |

**推荐**：自定义 StreamLanguage（与 mermaid-lang.ts 架构一致）

### 高亮规则

| Token | 颜色 | 示例 |
|-------|------|------|
| 命令 `\xxx` | 蓝色 #569cd6 | `\frac`, `\alpha`, `\begin` |
| 环境名 | 绿色 #4ec9b0 | `{aligned}`, `{cases}` |
| 数学运算符 | 白色加粗 | `+`, `-`, `=`, `<`, `>` |
| 上下标 | 橙色 #ce9178 | `^{2}`, `_{i}` |
| 花括号组 | 灰色 #808080 | `{`, `}` |
| 注释 | 绿色斜体 | `% comment` |
| `\text{}` 内文本 | 普通色 | `\text{if }` |

## 六、入口设计

### 6.1 全屏按钮位置

在 mathBlock 的 inline 编辑模式 header 中增加全屏按钮：

```
┌──────────────────────────────────────┐
│ ∑ Block equation              [⛶]   │  ← header 右侧增加全屏按钮
├──────────────────────────────────────┤
│ LaTeX 代码区                          │
├──────────────────────────────────────┤
│ 实时预览                              │
└──────────────────────────────────────┘
```

点击 `[⛶]` → 打开全屏编辑器，内容同步。

### 6.2 键盘快捷键

- 在 mathBlock 编辑模式中，`Cmd+Enter` → 打开全屏编辑器（类似 IDE 的"在编辑器中打开"）

## 七、与 Mermaid 全屏编辑器的差异

| 维度 | Mermaid | Math |
|------|---------|------|
| 渲染引擎 | mermaid.render() | katex.render() |
| 语法高亮 | mermaid-lang.ts | latex-lang.ts |
| Toolbar 左侧 | 模板 + 主题 + 方向 | 符号面板 + 环境模板 |
| 导出格式 | PNG/SVG（图表） | PNG/SVG（公式） |
| 预览区交互 | 缩放浏览（图大） | 滚动浏览（公式长） |
| debounce | 300ms（渲染重） | 150ms（KaTeX 快） |

## 八、实现顺序

### Phase 1 — 基础全屏编辑器
1. 提取 Mermaid 全屏编辑器的通用框架（overlay + 分屏 + CodeMirror + 状态栏）
2. 创建 `latex-lang.ts` 语法高亮
3. 替换渲染引擎为 KaTeX
4. mathBlock header 增加全屏按钮
5. 关闭时同步回 ProseMirror

### Phase 2 — 符号面板 + 环境模板
6. 符号面板 UI（分类 Tab + 网格）
7. 环境模板下拉
8. 点击插入到 CodeMirror 光标位置

### Phase 3 — 导出 + 增强
9. PNG/SVG 导出（复用 exportSvgAsPng/exportSvgAsFile）
10. 自定义宏支持
