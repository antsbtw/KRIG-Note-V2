# eBook UI 一致性评估（PDF / EPUB）（2026-04-09）

## 范围
- PDF（FixedPage）与 EPUB（Reflowable）模式下的 UI/交互一致性
- 参考文件：`src/plugins/ebook/components/EBookView.tsx`、`FixedPageContent.tsx`、`ReflowableContent.tsx`、`EBookToolbar.tsx`

## 发现（按严重度排序）
1) **工具栏功能暴露不一致**  
   - PDF 模式有缩放/适宽/页码跳转；EPUB 模式主要章节跳转/字体大小，缺少显式“页”概念，按钮布局共用但含义不同。用户在两种模式下切换时心理模型不一致。  

2) **搜索与书签体验差异**  
   - PDF 搜索结果基于页面；EPUB 搜索基于 CFI/章节，UI 入口相同但反馈形式/跳转单位不同，缺少模式提示。书签在 PDF 用页码，在 EPUB 用 CFI/章节，列表展示及跳转方式未统一格式化。  

3) **标注/选取交互仅覆盖 EPUB**  
   - 颜色选择器、selection 监听仅在 Reflowable/EPUB 流程中可用，PDF 模式缺少等价标注能力或清晰告知“不支持”。  

4) **进度与页码显示语义不一致**  
   - PDF 显示当前页/总页，EPUB 显示章节+百分比；工具栏未用模式标签区分，易混淆。  

5) **快捷键与命令差异未提示**  
   - Cmd+F/D、箭头键在 EPUB 被用于章节跳转；在 PDF 模式行为可能不同或无效，缺少模式化提示与帮助入口。  

## 改进建议
- 工具栏模式化：在 toolbar 显示当前渲染模式（PDF/EPUB），按模式切换显示相应控件；保持按钮数量与布局一致，但文案/tooltip 清晰标注作用单位（页/章节/百分比）。 
- 搜索与书签统一抽象：搜索结果、书签项展示统一包含“类型 + 定位字符串”（例如 `Page 12` 或 `Chapter • CFI`），跳转逻辑在适配层处理。 
- 标注能力声明：为 PDF 增加最小标注支持或显式“当前模式不支持标注”，并在 UI 中提示。 
- 进度显示分层：在 toolbar 以“进度”组件抽象，内部根据模式渲染 页/总页 或 章节/百分比，视觉保持一致。 
- 快捷键提示：在帮助/tooltip 中按模式列出快捷键，确保用户在切换模式后可预期行为。
